# Clickbait-Detection-Using-Deep-Transfer-Learning
A complete implementation of Clickbait Detection using deep learning, and transformer-based models (BERT, RoBERTa, ALBERT), trained on a Kaggle clickbait dataset with explainability using LIME.
Source: Kaggle Clickbait Dataset
Contains labeled examples of:
Clickbait
Not Clickbait
Note: The dataset is not included directly in this repository due to licensing.
